## 🧞 Quick Start(Local)

### 1. Clone & Install Packages

```bash

git clone https://github.com/prgrms-web-devcourse/Team-Meoguri-Linkocean-FE.git

npm install

```

### 2. Run 

```bash
npm run dev
```
Then, you can access to http://localhost:3000
